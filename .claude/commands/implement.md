# /implement

Goal: execute an agreed plan with disciplined, incremental edits.

Workflow:
1. Reconfirm the approved scope.
2. Modify the minimum code necessary for the target behavior.
3. Reuse existing abstractions and nearby patterns.
4. Verify each meaningful step before moving on.

Output:
- Focused implementation summary.
- Files changed.
- Validation performed.
- Any follow-up work that remains outside scope.
