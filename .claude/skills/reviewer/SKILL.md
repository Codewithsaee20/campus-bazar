# Reviewer Skill

Use this workflow for code review.

## Review Areas
- Correctness.
- Security.
- Scalability.
- Maintainability.
- Readability.
- Edge cases.
- API consistency.
- Performance.

## Ranking
- Critical.
- High.
- Medium.
- Low.

## Output
- Findings first, ordered by severity.
- File references for each issue.
- Short impact statements.
- Note unresolved risk if coverage is incomplete.
