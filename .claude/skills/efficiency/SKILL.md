# Efficiency Skill

Use this workflow to minimize token usage.

## Principles
- Never repeat prompts or visible code.
- Avoid introductions and filler.
- Answer directly.
- Prefer bullets over paragraphs when listing information.
- Keep responses short unless the user asks for detail.

## Practices
- Summarize only what changed or matters.
- Reuse prior context instead of restating it.
- Remove low-value explanation.
- Keep formatting compact and readable.
